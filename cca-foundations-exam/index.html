What Is the CCA Foundations Exam?

The Claude Certified Architect (CCA) Foundations exam is Anthropic's official certification for professionals who design, build, and deploy systems powered by Claude. It validates your ability to architect production-grade AI applications across five core domains.

Our full-length CCA Foundations exam simulation replicates the real testing conditions — timed at 120 minutes with 60 scenario-based questions weighted across all five domains.

The Five CCA Exam Domains

Each domain carries a specific weight in the overall score, matching the distribution used in the CCA certification:

1. Agentic Architecture (27%)

Design patterns for autonomous AI agents, orchestration strategies, multi-agent systems, and reliability patterns like SPIDER and retry loops.

2. Claude Code Configuration (20%)

CLAUDE.md files, project configuration, slash commands, permission models, and MCP server integration within Claude Code workflows.

3. Prompt Engineering (20%)

System prompt design, chain-of-thought techniques, few-shot patterns, role prompting, and the PRECISE framework for structured prompt construction.

4. Tool Design & MCP (18%)

Model Context Protocol server architecture, tool schema design, resource management, transport layers, and security considerations for tool-use patterns.

5. Context Management (15%)

Token budget strategies, caching with ephemeral and persistent patterns, multi-turn conversation design, and the CALM framework for context optimization.

For a deeper walkthrough of each domain — including worked scenarios and the traps that catch candidates — see our domain-by-domain breakdown, or jump straight to Domain 1: Agentic Architecture & Orchestration, the largest section of the exam.
